# Image-Segmentation
This project focuses on Image Segmentation, a computer vision technique used to divide an image into meaningful regions.The aim of this project is to separate objects or regions in an image based on pixel characteristics, making it easier to analyze and process visual data.

# How to Run the Image Segmentation Project

1.Download the project as a ZIP file

2.Extract the file and open the project folder

3.Make sure Python is installed on your system

4.Install the required libraries
pip install -r requirements.txt

5.Place the input images inside the images folder

6.Run the script
python main.py

7.The segmented output will be displayed and saved in the output folder.
