# Image-Segmentation
This project implements an **image segmentation system using the U-Net architecture**, deployed as a **web application** using **FastAPI**. Users can upload an image through a web interface, and the system returns a **segmented mask** generated by a trained deep learning model.

## Installation & Setup

### 1️.Clone the repository

git clone https://github.com/your-username/Image-Segmentation-unet.git
cd Image-Segmentation-unet

### 2️.Create & activate virtual environment
py -3.10 -m venv .venv
.venv\Scripts\activate

### 3️.Install dependencies
pip install fastapi uvicorn tensorflow numpy opencv-python pillow jinja2 python-multipart

## How to Run the Project
### Start the FastAPI server
cd Backend
python -m uvicorn main:app --reload

### You should see:

Uvicorn running on http://127.0.0.1:8000

## Access the Application

->Web App: http://127.0.0.1:8000

->API Docs (Swagger): http://127.0.0.1:8000/docs

## How It Works

1.User uploads an image through the frontend

2.Image is sent to the FastAPI backend

3.The trained U-Net model processes the image

4.A segmentation mask is generated

5.The mask is returned and displayed on the UI
